# beep

beep - текстовая игра, управляющаяся одной кнопкой. В этой игре можно пищать, а можно не пищать.

## Управление

- **пробел:** пищать
- **s:** ускорить текст

## Зависимости

beep использует фреймворк [LÖVE][LOVE] и требует для запуска версию LÖVE 0.10.x.
